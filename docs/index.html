How It Works

1
Feed it data
Any structured dataset — CSV, TSV, database export. No schema design, no field labeling, no configuration.
2
It discovers structure
Greedy pair-reduction finds which fields best disambiguate records, in what order. The result is a discriminator ladder — the data's own structural fingerprint.
3
You get insights
The ladder tells you what matters, the speedup tells you how structured your data is, and anomalies tell you what doesn't fit. Five capabilities from one algorithm.
